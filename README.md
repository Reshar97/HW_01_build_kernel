Собранное ядро Linux

Этот репозиторий содержит исходные коды, конфигурацию и готовый пакет ядра Linux, собранного для окружения Debian 12 на виртуальной машине QEMU/KVM.
Структура проекта

    <kernel_version>.deb — готовый пакет ядра для установки.
    .config — файл конфигурации, использованный при сборке.
    README.md — описание проекта.

Краткое описание

Проект демонстрирует процесс сборки и настройки ядра Linux. В качестве основы используется Debian 12, а виртуализация реализована с помощью QEMU/KVM. Конфигурация ядра оптимизирована и подготовлена для отладки.
Содержимое

    Скомпилированное ядро в виде .deb-пакета.
    Файл конфигурации .config.

Использование

    Установите пакет ядра стандартными средствами Debian.
    Перезагрузите систему и выберите новое ядро в меню загрузки.
    Проверьте работоспособность системы.

Примечания

    Для сборки использовались исходники ядра и окружение Debian 12.
    Конфигурация ядра приведена в файле .config.

Если потребуется подробная инструкция по сборке или использованию, её можно добавить по запросу.
